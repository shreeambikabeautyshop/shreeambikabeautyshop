# Shree Ambika Beauty Shop 🌸

**Premium Beauty Products eCommerce Website**

> 100% Original Products | Top Brands | Best Prices | WhatsApp Ordering

---

## 🚀 Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** React Icons
- **Fonts:** Poppins + Playfair Display (Google Fonts)
- **Deployment:** Vercel

## ✨ Features

- Responsive hero image slider with auto-play
- SEO optimized (metadata, OpenGraph, Twitter Cards, structured data)
- WhatsApp ordering integration
- Category grid with 9 categories
- Top brands marquee
- Trust & comparison section
- Social media connect section
- Floating WhatsApp button
- Fast loading with Next.js Image optimization

## 🏃 Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 📦 Build & Deploy

```bash
npm run build
npm start
```

## 🌐 Deploy on Vercel

1. Push to GitHub
2. Import repo on [vercel.com](https://vercel.com)
3. Click Deploy — done!
